# Генератор отчётов по времени (Streamlit)

Простое приложение для загрузки Excel-файлов и формирования отчётов по проектам и специалистам.

## Как запустить локально

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Деплой на Streamlit Cloud
1. Создайте новый публичный репозиторий на GitHub  
2. Загрузите туда файлы: `app.py`, `requirements.txt`, `README.md`  
3. Перейдите на [Streamlit Cloud](https://share.streamlit.io)  
4. Выберите **New app**, укажите свой репозиторий и `app.py`  
5. Нажмите **Deploy**
